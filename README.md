# Watchme - موقع تكميلي للمكملات الغذائية

موقع متخصص في عرض وتقديم المكملات الغذائية بمختلف أنواعها مع معلومات تفصيلية عن كل منتج.

## المميزات
- 📦 عرض متنوع للمكملات الغذائية
- 🔍 بحث متقدم عن المنتجات
- ℹ️ معلومات تفصيلية عن كل منتج
- 🛒 سلة تسوق
- 👤 حساب شخصي
- 💬 تقييمات والتعليقات

## التكنولوجيا المستخدمة
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js / Express
- **Database**: MongoDB
- **Styling**: Bootstrap / Tailwind CSS

## البدء السريع

### المتطلبات
- Node.js v14+
- npm أو yarn
- MongoDB

### التثبيت
```bash
# استنساخ المستودع
git clone https://github.com/hassanmihammed274/Watchme.git

# الدخ��ل للمجلد
cd Watchme

# تثبيت الحزم
npm install

# تشغيل الخادم
npm start
```
